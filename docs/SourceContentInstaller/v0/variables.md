# Variables

The following variables are autowritten to `variables.json` and define the path where SourceContentIntaller installed Source games content.

## Counter-Strike: Source

| Name | Description |
| ---- | ----------- |
| cstrike_content_path | Counter-Strike: Source (base) content install path |
| cstrike_maps_content_path | Counter-Strike: Source (maps) content install path |
| cstrike_gamedir | Counter-Strike: Source game directory |
