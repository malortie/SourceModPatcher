## Prerequisites

## Installation

1. Go to Library &gt; Tools
2. Install Source SDK Base 2013 Singleplayer
3. Right-click on Source SDK Base 2013 Singleplayer &gt; Properties
4. Go to Betas
5. Select *upcoming - upcoming*
6. Restart Steam
7. Launch Source SDK Base 2013 Singleplayer
